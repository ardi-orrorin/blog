# Branch

## Overview
- 브랜치는 독립적으로 어떤 작업을 진행하기 위한 개념이다.
- 브랜치를 통해 독립적으로 작업을 진행하고, 작업이 완료되면 브랜치를 합치는 방식으로 작업을 진행한다.

## New Branch
- 브랜치 생성
```bash
$ git branch <branch-name>
```

- 브랜치 목록 확인
```bash
$ git branch
```

- 브랜치 이동
```bash
$ git checkout <branch-name>
```

- 원격 브랜치 가져오기(fetch)
```bash
$ git fetch origin
```
