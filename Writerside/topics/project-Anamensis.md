## Tech Skills

### Client
- Language : Typescript
- PackageManager : NPM
- Build : TurboPack
- Framework : NEXTJS
- Libraries
    - Date : momentJs
    - fetchAPI : axios
    - 스타일 : tailwind
    - 문법 및 오류 체크 : ESLint
- Test
    - 미정


### Server
- Language : Java 17
- PackageManager & Build : Gradle
- Framework : Springboot
- Libraries
    - Webflux
    - SpringSecurity
    - Mybatis
    - SpringValidation
- Test
    - Junit

### Database
